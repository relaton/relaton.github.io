== Cititng OMG standards

=== Prefix

The prefix for OMG standards is `OMG`:

[example]
`OMG AMI4CCM`

=== Version

Version can be specified in the end of the citation:

[example]
----
OMG AMI4CCM 1.0
OMG AMI4CCM 1.1
OMG API4KP 1.0 beta 2
----

Without version, the latest version is assumed.

OASIS standards are available in the
https://www.relaton.org/relaton-data-oasis/[OASIS Standards index].

Prefix for all OASIS standards is `OASIS`.

[example]
`OASIS AkomaNtosoCore-v1.0-Pt1-Vocabulary`

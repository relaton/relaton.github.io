== Prefix

The prefix `IHO` identifies it as an IHO document.

[example]
`IHO B-4`

== Edition

Some IHO standards have multiple editions:

[example]
[source]
----
IHO S-100 1.0.0
IHO S-100 2.0.0
----


== Volume

Some IHO standards have multiple volumes:

[example]
[source]
----
IHO P-1/21
IHO P-1/22
----

== Annex

[example]
[source]
----
IHO S-65AnnexA 1.0.0
----
